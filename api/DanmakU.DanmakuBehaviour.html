    Remarks

    A given DanmakuBehaviour manages the lifetime of any of it's created DanmakuSets. Sets created with CreateSet(DanmakuPrefab) will automatically be disposed of when the component is destroyed. This means any bullets fired by these sets will be destroyed when the component is destroyed.
